node-thumbnail
==============

thumbnail all the things

usage:

    thumb [options] source/path dest/path

options:

    -h, --help
    -v, --version

    -s SUFFIX, --suffix SUFFIX
    -d, --digest
    -t TYPE, --hashing-type TYPE

    -w, --width

    -c NUM, --concurency NUM

installation
------------

    $ brew install imagemagick
    $ npm install node-thumbnail
